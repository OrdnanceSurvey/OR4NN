# OR4NN - Object Range for Scalable K-NN Join

OR4NN is a conceputal framework for Efficient Scalable KNN Query on Metric Data on distributed/Parallel Computing Platform. This repo contains an experimental implementation of OR4NN for 2D geosptial data.

<img width="532" height="532" alt="image" src="https://github.com/user-attachments/assets/b9849948-4322-4190-b8b6-c44a7792c33b" />

Please read the technical report and the draft submitted to GISRUK2026 for detailed information on OR4NN.

You will require Apache Sedona installed on your cluster to run the notebooks. Please refer to the user guide for instructions.

A patent application reflecting the work in this repo has been filed (UK Patent Office No. 2411135.3).
